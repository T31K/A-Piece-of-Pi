# WORK IN PROGRESS
## To Build


### Clone Repo
```sh
$ git clone https://github.com/T31K/A-Piece-of-Pi
```

### Install Server Dependecies
```sh
$ cd server
$ npm install 
$ npm start
```

### Install Client Dependecies
```sh
$ cd client
$ npm install
$ npm start
```
